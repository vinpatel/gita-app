**Theme-adaptive accent derivation (per D-07):**
```css
/* Light mode chapter accent (default) */
:root {
  --chapter-accent: oklch(55% 0.14 var(--chapter-hue));
  --chapter-accent-dim: oklch(55% 0.14 var(--chapter-hue) / 0.15);
}

/* Dark mode: brighter, more saturated */
html.dark {
  --chapter-accent: oklch(72% 0.18 var(--chapter-hue));
  --chapter-accent-dim: oklch(72% 0.18 var(--chapter-hue) / 0.15);
}

/* Sepia mode: warm-shifted +10 degrees */
html.sepia {
  --chapter-accent: oklch(58% 0.12 calc(var(--chapter-hue) + 10));
  --chapter-accent-dim: oklch(58% 0.12 calc(var(--chapter-hue) + 10) / 0.15);
}
```

NOTE: --chapter-accent is NOT set globally to a specific hue — it only resolves when a .chapter-N class is present on an ancestor element. This is correct per Pitfall 5 from RESEARCH.md. Pages must apply .chapter-N class on body or main to activate.

Also add utility classes for chapter accent usage (per D-06 — borders, stripes, badges only):
```css
@layer utilities {
  .chapter-accent-border { border-color: var(--chapter-accent); }
  .chapter-accent-stripe { background-color: var(--chapter-accent); }
  .chapter-accent-badge { background-color: var(--chapter-accent-dim); color: var(--chapter-accent); }
  .chapter-accent-text { color: var(--chapter-accent); }
}
```
