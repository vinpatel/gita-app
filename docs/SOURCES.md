# Canonical Sources

## Verse Numbering Standard

The **Gita Press (Gorakhpur)** edition of the Bhagavad Gita is the canonical source for verse numbering and chapter structure in this project.

This choice establishes a single authoritative reference for the 700-verse, 18-chapter structure used throughout all content, data schemas, and API design.

## Legal Status of the Sanskrit Text

The original Sanskrit text of the Bhagavad Gita is **public domain**. The Delhi High Court (2024) affirmed that original Sanskrit religious scriptures are in the public domain and not subject to copyright. Gita Press does not hold copyright on the source Sanskrit verses — their numbering standard is a reference choice, not a licensing dependency.
